SAGrader

SAGrader is a modern Perl web application backed by MySQL and Redis. It is a fully featured learning management system used by 4,000 students a year.

Instructors can collaborate with our content developers to create their own custom writing assignments from short answer to full length essays. Students submit to these assignments and receive instant, automated, content-based feedback tailored to their response. Students are encouraged to revise to fully master the topic.

SAGrader's core technology uses natural language processing to do qualitative text analysis of our student's writing. This allows SAGrader to give very detailed, specific feedback instantly to students.

Some of the features I've worked on are described below.

Created a backend service and UI to provide content developers with analytics about how questions are performing.

This tool analyzes how students perform on each concept in the question so content developers can identify which concepts students struggle with.

Developed a quick creation tool for content developers to rapidly build a skeleton semantic network, significantly speeding up question development.

A simplified version can be demoed here.

Redesigned the format of SAGrader's automated feedback, condensing the information and making it easier for students to understand.

This involved completely re-imagining the way we traverse the graph of the student's knowledge and the way we present it to the student.

Detected and moved many long running tasks into backend services.

This included tasks such as: calculating course scores, downloading gradebooks, data exporting, and instructor test grading of assignments.

Created multiple simple UIs to allow instructors to manage large amounts of student data on a single screen.

These features enable instructors to actually respond to student concerns such as challenging their grade and having their response reviewed by their instructor.

Redesigned the instructor assignment UI to organize and clarify an overwhelming number of options and restrictions.

Retrofitted the entire student UI with a responsive layout. Enabling students to view their course and submit on any device.

I've also fixed 325 tickets, written 7,982 tests, and single-handedly run SAGrader for three years.
